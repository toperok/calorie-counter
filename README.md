# Предсказание калорийности блюд (Multi-modal Regression)

Проект по созданию нейросети, которая предсказывает калорийность блюда на основе двух типов данных: **фотографии** и **текстового описания** его состава. Это решение объединяет компьютерное зрение (CV) и обработку естественного языка (NLP).

## 📌 Описание задачи

Ваша задача — реализовать и обучить мультимодальную модель, которая предсказывает числовое значение (калории). Модель может быть полезна в fitness-трекерах и health-приложениях для автоматизации контроля питания.

### Этапы работы:

1. **Исследовательский анализ (EDA)**: Изучение распределения данных, визуализация и выбор стратегии аугментации.
2. **Реализация пайплайна**: Создание модульной структуры кода в `.py` файлах.
3. **Обучение**: Запуск тренировочного цикла (40 эпох) с использованием GPU и логированием результатов.
4. **Валидация**: Тестирование модели, расчет метрик (MAE, RMSE, MAPE) и анализ «тяжелых» примеров.
